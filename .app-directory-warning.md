# ⚠️ 警告: このディレクトリを作成しないでください

このリポジトリでは、**ルート直下に `app/` ディレクトリを作成してはいけません**。

## 理由

- Next.js App Router は `src/app/` を使用しています
- ルート直下に `app/` を作成すると、`src/app/` をシャドウしてしまい、404エラーが発生します
- Next.js は `app/` と `src/app/` の両方が存在する場合、`app/` を優先します

## 正しい構成

```
プロジェクトルート/
├── src/
│   └── app/          ← これが唯一の App Router ディレクトリ
│       ├── page.tsx
│       ├── layout.tsx
│       └── ...
├── package.json
└── next.config.js
```

## 誤った構成（404の原因）

```
プロジェクトルート/
├── app/              ← ❌ これを絶対に作らないでください
│   └── page.tsx
├── src/
│   └── app/         ← これがシャドウされて使われなくなります
│       └── page.tsx
└── ...
```

## もし誤って作成してしまった場合

1. `app/` ディレクトリを削除してください
2. `git status` で変更を確認
3. `git add app/` で削除をステージング
4. `git commit -m "Remove root app directory"` でコミット
5. `git push` でプッシュ
